### props.checkDeprecatedProps(props, deprecatedNames, callback)
Checks for deprecated props and calls a callback for each match

**Kind**: static method of [`props`](#utils.module_props)  

| Param | Type | Description |
| --- | --- | --- |
| props | `object` | The current props object |
| deprecatedNames | `Array.<string>` | Array of prop names to check |
| callback | `function` | Function called for each match, receiving the prop name |
