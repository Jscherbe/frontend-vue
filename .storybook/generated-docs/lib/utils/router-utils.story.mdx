<a name="module_router-utils.getRouteTitle"></a>

### router-utils.getRouteTitle(route, [currentRoute]) ⇒ `string` \| `undefined`
Resolves a route's title from its meta.
- If `meta.title` is a function, it's called with the `currentRoute` (or the route itself).
- Otherwise, `meta.title` is returned.
This function is the single source of truth for resolving titles from route configuration.

**Kind**: static method of [`router-utils`](#module_router-utils)  
**Returns**: `string` \| `undefined` - The resolved title, or undefined if not found.  

| Param | Type | Description |
| --- | --- | --- |
| route | `object` | The route or route match object. |
| [currentRoute] | `object` | The current route from `useRoute()`, passed to functional titles. |

<a name="module_router-utils.$createSectionMenu"></a>

### router-utils.$createSectionMenu(route, options) ⇒ [`Array.<RouteMenuItem>`](#module_router-utils..RouteMenuItem)
For a given $route will return all it's children as menu items,
using the route's meta.title property for the title. This is for sections only (routes with children)
- Useful for dynamic menus (menus within some unknown section) where you don't want to write static paths

**Kind**: static method of [`router-utils`](#module_router-utils)  
**Returns**: [`Array.<RouteMenuItem>`](#module_router-utils..RouteMenuItem) - Array of menu items  

| Param | Type | Description |
| --- | --- | --- |
| route | `Object` | Actual $route object |
| options | `Object` | Options |
| options.parent | `Object` | Route parent object, defaults to parent of route |
| options.includeIndex | `Boolean` | Include the parent/index in the menu items ie path: "" |
| options.item | `Object` | Options for createMenuItem |

<a name="module_router-utils.$createBreadcrumb"></a>

### router-utils.$createBreadcrumb(route) ⇒ `Array.<Object>`
For a given $route, this will generate a breadcrumb trail.
It iterates through `route.matched` to build the trail.
- Falls back to `meta.title` (string or function).
- Skips routes where `meta.breadcrumb` is set to `false`.
- Avoids duplicate crumbs for nested routes with empty paths.

**Kind**: static method of [`router-utils`](#module_router-utils)  
**Returns**: `Array.<Object>` - An array of breadcrumb items with title, to, and current properties.  

| Param | Type | Description |
| --- | --- | --- |
| route | `Object` | The Vue Router `$route` object. |
