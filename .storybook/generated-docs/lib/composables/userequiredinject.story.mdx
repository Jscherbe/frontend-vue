### useRequiredInject.useRequiredInject(key) ⇒
Injects a dependency from a plugin (or other required inject) and throws an error if it's not available.

**Kind**: static method of [`useRequiredInject`](#module_useRequiredInject)  
**Returns**: The injected value.  

| Param | Type | Description |
| --- | --- | --- |
| key | `string` | The injection key (e.g., 'uluBreakpointManager'). |
